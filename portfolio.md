# AI スライド自動生成ツール

## 概要

テーマを入力するだけで、AI がプレゼンテーションの構成・文章・イラストをすべて自動生成し、PowerPoint ファイルとしてダウンロードできる Web アプリケーション。

## できること

- テーマと枚数を入力するだけでスライドを自動生成
- 各スライドに AI イラストを自動挿入（Google Imagen 4）
- プレビュー画面で生成されたスライドを確認
- 修正指示を入力して何度でも内容を改善
- 満足したら PowerPoint (.pptx) としてダウンロード

## 使用技術

| カテゴリ | 技術 |
|---------|------|
| Backend | Python 3.12 / FastAPI / Uvicorn |
| Frontend | React 19 / TypeScript / Tailwind CSS / Vite |
| AI (テキスト) | Google Gemini API（gemini-2.5-flash） |
| AI (画像) | Google Imagen API（imagen-4.0-generate-001） |
| スライド生成 | python-pptx / Pillow |
| インフラ | Docker / Docker Compose / AWS CDK（ECS Fargate） |
| テスト | pytest / Playwright / httpx（ユニット 28件 / インテグレーション 14件 / E2E 9件） |

## 設計

- DDD + クリーンアーキテクチャ（Domain → Application → Infrastructure → Presentation）
- Result 型パターンでエラーを明示的に処理
- DI コンテナによる依存性注入
- AOP（横断的関心事）でエラーハンドリングをミドルウェアに集約

## 導入効果

| | 導入前 | 導入後 |
|---|---|---|
| 資料作成時間 | 2〜3時間/件 | 5分/件 |
| 資料の品質 | 担当者によりバラつき | フォーマット統一 + AI イラスト付き |
| 商談準備 | 資料作成に追われる | 提案内容の検討に集中 |
