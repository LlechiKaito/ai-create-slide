# スライド作成手順メモ

## 準備
1. Docker Desktop を起動する
2. ターミナルでプロジェクトディレクトリに移動する
3. `.env` に Gemini API キーを設定する（`GEMINI_API_KEY=your_key`）
4. Claude Code を起動する

## 画像を生成する
- Gemini API（imagen-4.0-fast-generate-001）で画像を生成
- 1枚 $0.02（課金が必要。無料枠は2025年12月に廃止済み）
- 実行コマンド：
  ```bash
  docker compose run --rm generate "プロンプト" -o output/assets/ファイル名.png --aspect-ratio 16:9
  ```
- アスペクト比の選択肢：`1:1` `3:4` `4:3` `16:9` `9:16`
- スライドに配置するサイズに合ったアスペクト比で生成すること（引き伸ばし注意）

## スライドの構成を決める
- テーマを決める
- 1枚1メッセージでスライド構成を書き出す
- 今回は7枚構成：タイトル → 経歴・スキル → 今やっていること → アニメ → ボカロ → 仕事の特徴 → まとめ

## AIに指示を出す（PPTX生成）
- `create_pptx.py` を python-pptx で作成
- 必ず伝えること：
  - 各スライドに表示するテキストの具体的な内容
  - 色の指定（背景・文字・アクセント）
  - 画像の配置とアスペクト比
  - フォントサイズや太字の指定
- 実行コマンド：
  ```bash
  docker compose run --rm pptx
  ```
- 出力先：`output/slides/self_introduction.pptx`

## 修正する
- PowerPoint で開いて1枚ずつ確認
- 修正指示は「スライドN の〇〇を変えて」と具体的に
- テキストの被りは特に注意（長いテキストは改行で対応）
- 「、」は使わず改行で区切る

## PNG版スライド生成（Pillow）
- 初期に作成した PNG 版も残っている
- 実行コマンド：
  ```bash
  docker compose run --rm slides
  ```
- 出力先：`output/slides/slide_1_title.png` 〜 `slide_5_summary.png`

## プロジェクト構成
```
ai-create-slide/
├── .env                  # APIキー（gitignore対象）
├── .env.example          # APIキーのテンプレート
├── .gitignore
├── Dockerfile            # Python 3.12 + 日本語フォント
├── docker-compose.yml    # generate / slides / pptx サービス
├── requirements.txt      # google-genai / Pillow / python-pptx
├── generate_image.py     # Gemini API 画像生成スクリプト
├── create_slides.py      # Pillow PNG スライド生成
├── create_pptx.py        # python-pptx PPTX スライド生成
└── output/
    ├── assets/           # 生成した画像素材
    └── slides/           # 生成したスライド
```

## 自分用メモ
- テキストが被るときはフォントサイズを下げるか行を分割する
- カード内のテキスト行数が多いときは card_h を増やす
- ラベル:値 の揃えは別テキストボックス（右揃え+左揃え）で対応
- 画像配置は生成時のアスペクト比と同じ比率で width/height を指定する
- 「、」は使わず改行で対応する方針
- 配色：白背景 / ダークグレー文字 RGB(50,50,50) / オレンジアクセント RGB(240,130,40)
